## 项目简介

这是 [李苑飞 (lyf2nb)](https://github.com/lyf2nb) 个人维护的软件下载导航站。  
互联网上充斥着大量捆绑软件、钓鱼镜像和假冒安装包，本站的目标只有一个：**直达官方，省去甄别烦恼**。

收录范围包括浏览器、开发工具、办公软件、设计创意、影音娱乐、系统工具等主流分类，所有链接均指向软件官网或官方发布渠道。

> ⚠️ **本站为 AI 辅助开发**，如发现信息有误，欢迎在作者 GitHub 任意仓库提交 Issue，或发邮件至 lyf_fbq@163.com

## 功能特性

- **分类导航**：侧边栏按软件类型分类，一栏总览全站收录
- **实时搜索**：顶部搜索框即时过滤，输入软件名立刻定位
- **用户投票**：每条链接支持点赞 / 踩，帮助社区筛选更优质的资源
- **社区提交**：任何人都可以提交新软件或新分类，经管理员审核后上线
- **管理审核**：内置审核面板，管理员可一键批准或拒绝待审条目，支持角标提示待审数量
- **深色 / 浅色主题**：一键切换，跟随个人偏好
- **纯静态前端**：无需注册登录，打开即用

## 技术栈

| 层面 | 技术 |
|------|------|
| 前端 | 原生 HTML / CSS / JavaScript（无框架） |
| 数据 | `/api/data` REST 接口（后端自行部署） |
| 存储 | 投票记录存于浏览器 `localStorage` |
| 部署 | 任意静态托管 / Node.js 后端 |

---

## API 接口一览

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/data` | 获取全部软件与分类数据 |
| POST | `/api/vote/:swId/:linkId/:type` | 投票（up / down） |
| POST | `/api/software/:swId/cracked` | 提交新链接（进审核队列） |
| DELETE | `/api/software/:swId/cracked/:linkId` | 删除链接（需管理员密码） |
| GET | `/api/pending` | 获取待审核列表（需管理员密码） |
| POST | `/api/review/:id` | 审核通过 / 拒绝（需管理员密码） |
